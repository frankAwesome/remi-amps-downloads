REMI Amps — downloads
=====================

The site's download buttons link to the LATEST GitHub Release, so they always
serve the newest installer with no HTML changes:

  Windows : https://github.com/frankAwesome/remi-amps-downloads/releases/latest/download/REMI-Amps-Windows-Setup.exe
  macOS   : https://github.com/frankAwesome/remi-amps-downloads/releases/latest/download/REMI-Amps-macOS.pkg

This folder is just a local staging copy (git-ignored): the Windows build script
drops REMI-Amps-Windows-Setup.exe here so you can preview the site offline. The
canonical hosting is the GitHub Release.

To cut a release (and keep the asset names that the links depend on), see
../../installer/RELEASING.md.
