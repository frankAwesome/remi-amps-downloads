# REMI Amps — Downloads

Public installer downloads for **REMI Amps — Over the Edge**, a JUCE guitar amp +
multi-effects plugin (Standalone app **+** VST3 **+** Audio Unit).

## Download

Grab the latest installers from the [**Releases**](../../releases/latest) page:

| Platform | File |
|---|---|
| Windows 10 / 11 (64-bit) | `REMI-Amps-Windows-Setup.exe` |
| macOS 11+ (Intel & Apple Silicon) | `REMI-Amps-macOS.pkg` |

The download links always resolve to the newest release:

- <https://github.com/frankAwesome/remi-amps-downloads/releases/latest/download/REMI-Amps-Windows-Setup.exe>
- <https://github.com/frankAwesome/remi-amps-downloads/releases/latest/download/REMI-Amps-macOS.pkg>

> **Note:** these installers are currently unsigned. On **macOS**, right-click the
> `.pkg` → **Open** (or approve it under System Settings → Privacy & Security). On
> **Windows**, if SmartScreen warns, click **More info → Run anyway**.

The application source code is maintained in a separate private repository.
